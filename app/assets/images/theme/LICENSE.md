# Theme switch icons

The SVG icons in this directory are taken unmodified from **Bootstrap Icons**:

| File                  | Bootstrap Icon name | Used for theme state |
| --------------------- | ------------------- | -------------------- |
| `circle-half.svg`     | `circle-half`       | auto (follow OS)     |
| `sun-fill.svg`        | `sun-fill`          | light                |
| `moon-stars-fill.svg` | `moon-stars-fill`   | dark                 |

## Source

- Project: Bootstrap Icons — https://icons.getbootstrap.com/
- Repository: https://github.com/twbs/icons
- Individual icons:
  - https://icons.getbootstrap.com/icons/circle-half/
  - https://icons.getbootstrap.com/icons/sun-fill/
